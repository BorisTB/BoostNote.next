## Build

Don't forget to create a file named `.env` in the root of the project directory, or the build will fail.

### Web app

```sh
# Install deps
npm i

# Compile script
npm run build

# Deploy to now.sh
now
```

### Electron app

```sh
# Install deps
npm i

# Compile script
npm run build

# Package app
npm run pack
```
